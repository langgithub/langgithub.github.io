  2. д№җи§Ӯй”Ғ
    иҮӘж—Ӣй”Ғ
    йҒҝе…Қз”ЁжҲ·зәҝзЁӢе’ҢеҶ…ж ёзҡ„еҲҮжҚўзҡ„ж¶ҲиҖ—,ж— йңҖиҝӣе…Ҙйҳ»еЎһзҠ¶жҖҒжҢӮиө·
    еҸӮж•°и®ҫзҪ®пјҡ
    JDK1.6дёӯ-XX:+UseSpinningејҖеҗҜпјӣ
    -XX:PreBlockSpin=10 дёәиҮӘж—Ӣж¬Ўж•°пјӣ
    JDK1.7еҗҺпјҢеҺ»жҺүжӯӨеҸӮж•°пјҢз”ұjvmжҺ§еҲ¶пјӣ
    иҪ»йҮҸзә§й”Ғ
    иҪ»йҮҸзә§й”ҒжҳҜз”ұеҒҸеҗ‘жүҖеҚҮзә§жқҘзҡ„пјҢеҒҸеҗ‘й”ҒиҝҗиЎҢеңЁдёҖдёӘзәҝзЁӢиҝӣе…ҘеҗҢжӯҘеқ—зҡ„жғ…еҶөдёӢпјҢеҪ“з¬¬дәҢдёӘзәҝзЁӢеҠ е…Ҙй”Ғдәүз”Ёзҡ„ж—¶еҖҷпјҢеҒҸеҗ‘й”Ғе°ұдјҡеҚҮзә§дёәиҪ»йҮҸзә§й”Ғпјӣ
    еҒҸеҗ‘й”Ғ
    еҰӮжһңеңЁиҝҗиЎҢиҝҮзЁӢдёӯпјҢйҒҮеҲ°дәҶе…¶д»–зәҝзЁӢжҠўеҚ й”ҒпјҢеҲҷжҢҒжңүеҒҸеҗ‘й”Ғзҡ„зәҝзЁӢдјҡиў«жҢӮиө·пјҢJVMдјҡж¶ҲйҷӨе®ғиә«дёҠзҡ„еҒҸеҗ‘й”ҒпјҢе°Ҷй”ҒжҒўеӨҚеҲ°ж ҮеҮҶзҡ„иҪ»йҮҸзә§й”Ғ

жҖ»з»“пјҡ

еңЁжүҖжңүзҡ„й”ҒйғҪеҗҜз”Ёзҡ„жғ…еҶөдёӢзәҝзЁӢиҝӣе…Ҙдёҙз•ҢеҢәж—¶дјҡе…ҲеҺ»иҺ·еҸ–еҒҸеҗ‘й”ҒпјҢеҰӮжһңе·Із»ҸеӯҳеңЁеҒҸеҗ‘й”ҒдәҶпјҢеҲҷдјҡе°қиҜ•
иҺ·еҸ–иҪ»йҮҸзә§й”ҒпјҢеҗҜз”ЁиҮӘж—Ӣй”ҒпјҢеҰӮжһңиҮӘж—Ӣд№ҹжІЎжңүиҺ·еҸ–еҲ°й”ҒпјҢеҲҷдҪҝз”ЁйҮҚйҮҸзә§й”ҒпјҢжІЎжңүиҺ·еҸ–еҲ°й”Ғзҡ„зәҝзЁӢйҳ»еЎһ
жҢӮиө·пјҢзӣҙеҲ°жҢҒжңүй”Ғзҡ„зәҝзЁӢжү§иЎҢе®ҢеҗҢжӯҘеқ—е”ӨйҶ’д»–д»¬пјӣ

еҒҸеҗ‘й”ҒжҳҜеңЁж— й”Ғдәүз”Ёзҡ„жғ…еҶөдёӢдҪҝз”Ёзҡ„пјҢд№ҹе°ұжҳҜеҗҢжӯҘејҖеңЁеҪ“еүҚзәҝзЁӢжІЎжңүжү§иЎҢе®Ңд№ӢеүҚпјҢжІЎжңүе…¶е®ғзәҝзЁӢдјҡжү§
иЎҢиҜҘеҗҢжӯҘеқ—пјҢдёҖж—ҰжңүдәҶз¬¬дәҢдёӘзәҝзЁӢзҡ„дәүз”ЁпјҢеҒҸеҗ‘й”Ғе°ұдјҡеҚҮзә§дёәиҪ»йҮҸзә§й”ҒпјҢеҰӮжһңиҪ»йҮҸзә§й”ҒиҮӘж—ӢеҲ°иҫҫйҳҲеҖј
еҗҺпјҢжІЎжңүиҺ·еҸ–еҲ°й”ҒпјҢе°ұдјҡеҚҮзә§дёәйҮҚйҮҸзә§й”Ғпјӣ
