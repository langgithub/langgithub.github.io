ELF文件静态感染注入

http://www.520monkey.com/archives/561

要修改的位置

  1. ELF Header
    Start of program headers
    Number of program headers
  2. Program Headers

PT_LOAD: 描述程序加载时内存存储信息
动态节区 .dynamic(PT_DYNAMIC),包含了程序动态链接和依赖库信息（libhello.so）

typedef struct{
ELF32_Sword d_tag;
union{
ELF32_Word d_val;
ELF32_Addr d_ptr;
}
}

d_tag类型：dt_null,dt_needed,dt_strtab,dt_strsz

感染ELF文件注入实现：

1. 在dt_strtab指向的字符串中添加自定义so模块名称，将字符串表移动到文件末尾
2. 添加一个pt_load表，用于能够内存映射我们添加的字符串，phdr移动到文件末尾
3. 修改dt_strtab,dt_strsz,添加dt_needed
4. 修改ELF Header关于段头表的信息字段，偏移，大小
